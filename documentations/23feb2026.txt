2. What each part does
database.js → connects to MongoDB Atlas
userModel.js → defines user schema (username, email, password, timestamps)
userController.js → CRUD operations (getUsers, getUsersById, createUser, updateUser, deleteUser)
authController.js → register (create user + token) and login (check password + token)
hashPassword.js → middleware that hashes password before saving to DB
authMiddleware.js → middleware that verifies JWT token on protected routes
generateToken.js → utility that creates JWT token
userRoutes.js → CRUD routes, protected with JWT
authRoutes.js → register and login routes
